# Détection du Model Drift — Crédit Scoring
Projet de fin d'année — Licence Mathématiques Appliquées
Faculté des Sciences de Tunis — 2025/2026

## Description
Ce projet implémente un système de détection du model drift 
dans un modèle de crédit scoring (Random Forest) 
appliqué au contexte bancaire tunisien (UIB).
